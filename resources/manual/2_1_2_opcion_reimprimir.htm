2.1.2 Opcion Reimprimir

 
 
esta opcion permitira volver a imprimir una o varias etiquetas para una derminada orden de produccion
 
 
1. click en reimprimir
 
 
 
2. se abre la ventana modal  reimprimir codigos de barra
 
 
 
3. AREA  BUSCAR PRODUCTO
 
muestra por defecto el producto que se guardo en al ultima generacion de seriales, este campo se puede modificar , esto actualiza el codigo de producto en la tabla de seriales y ademas imprimira la etiqueta con el codigo del producto que el usuario tenga seleccionado en este momento, para modificar basta con dar click en el campo y escribir el nombre  o codigo del nuevo producto
 
 
 
4. AREA  GENERAR E IMPRIMIR
 
A. CAMPOS
 
los botones en gris estan deshabilitados y no se pueden modificar
 
producto: descripcion del producto segun produccion
Codigo pr: codigo del producto segun produccion
Total: cantidad de productos a fabricar segun la orden de produccion ( impreso + pendiente)
Impreso: cantidad de productos a los cuales se les ha generado serial y se ha impreso el codigo de barras
Pendiente: cantidad de productos a los cuales no  se les ha generado serial ni  se le  ha impreso el codigo de barras
          Imprimir: este campo tiene 3 opciones  (unico - rango - todos)
    
#Copias: en este campo se debe seleccionar el numero de impresiones que se van a realizar de cada                         etiqueta (si es 0 significa que solo se imprimira la etiqueta original sin copias, si es 1 significa que se                van a imprimir 2 etiquetas la orginal y 1 copia ....), cuando se reimprime una etiqueta se llevara un                     conteo del numero de reimpresiones que se hace por cada serial, este dato se podra visualizar en la                opion detalle  de la vista principal
Numero:  identificador unico para la orden de produccion (numero de orden)
Serial Inicial: indica el primer serial (10 digitos) desde donde se comenzara a reimprimir
   
Serial Final: indica el ultimo serial (10 digitos) que se reimprimira
 
 
B. CAMPO IMPRIMIR OPCION UNICO
 
si el campo imprimir es unico y seleccionamos un serial en el campo Serial Inicial , automaticamente se actualizar el campo Serial Final con el mismo numero de serial que se haya seleccionado en el campo Serial Inicial, esto hara que se imprimima solamente ese serial
 
 
 
C. CAMPO IMPRIMIR  OPCION RANGO
 
si el campo imprimir es rango , entonces debemos seleccionar un serial inicial en el campo Serial Inicial y un serial final en el campo Serial Final, con esto se imprimir los seriales que esten dentro de ese rango
 
 
 
 
D. CAMPO IMPRIMIR  OPCION TODOS
 
si el campo imprimir es todos y seleccionamos cualquier serial en el campo Serial Inicial , automaticamente se actualizara el campo Serial Inicial con el serial  minimo existente para la orden actual y tambien se actualizara el campo Serial Final con el serial maximo existente para la  orden actual
 
 
 
E. EJEMPLO
 
este ejemplo generara 2 seriales  (19090730001-19090730002) para el COM IBG ASTURIAS 4PTOS, e imprimira 4 etiquetas (2 originales y 2 copias)
 
 
F. IMPRESION
 
al dar click en el boton de impresion se guardaran los seriales y se imprimiran  las 4  etiquetas  (1 original y  1 copia), el numero de serie debera ir en consecutivo de 10 digitos, los 7 primero digitos corresponde al numero de orden de produccion , los ultimos 3 seran un consecutivo de acuerdo al total  de producto de esa orden
 
numero orden  = 1909073
consecutivo 1 = 1909073002
consecutivo 2 = 1909073003
